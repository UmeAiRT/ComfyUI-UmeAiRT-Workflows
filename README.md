## ðŸ“¥ Installation

### Method 1: Via ComfyUI Manager (Recommended)
1.  Open **ComfyUI Manager**.
2.  Click on **"Install via Git"**.
3.  Paste the URL of this repository: `https://github.com/UmeAiRT/ComfyUI-UmeAiRT-Workflows`
4.  Restart ComfyUI.

### Method 2: Manual Installation
1.  Navigate to your `ComfyUI/custom_nodes/` directory.
2.  Clone this repository:
    ```bash
    git clone [https://github.com/UmeAiRT/ComfyUI-UmeAiRT-Workflows.git](https://github.com/UmeAiRT/ComfyUI-UmeAiRT-Workflows.git)
    ```
3.  Restart ComfyUI.

## ðŸš€ How to use

Once installed, there are two ways to load the workflows:

### 1. The "Drag & Drop" Method
Navigate to the folder `custom_nodes/ComfyUI-UmeAiRT-Workflows/workflows/` on your computer.
Simply **drag and drop** any PNG image into the ComfyUI window. The workflow will load instantly.

### 2. Native Menu (ComfyUI V1 / Custom Scripts)
Since this pack follows the standard folder structure, if you are using the latest ComfyUI interface or extensions like `pysssss`, you can access the workflows directly inside the interface:
* Right-click on the canvas context menu.
* Look for the **"Workflows"** or **"Examples"** entry.
* Select **UmeAiRT**.
